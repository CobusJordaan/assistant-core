User Management

Open WebUI has its own built-in user system. When adding friends or family members:

Create user accounts inside Open WebUI Admin Settings — this is the proper way.
Use the Open WebUI role system to control what each user can do (user vs admin).
Do not create Linux system users for friends/kids.
Do not give friends/kids access to this admin dashboard.
