Open WebUI Setup

To connect Open WebUI to Draadloze AI:

  1. Go to Open WebUI Admin Settings → Connections
  2. Add a new OpenAI connection
  3. Set the URL to: http://172.18.2.195:{{ router_port }}/v1
  4. Set the API Key to the generated key above (airt_...)
  5. Save and verify — you should see draadloze-ai in the model list
  6. For non-admin users: set draadloze-ai as the default model and hide other models in user permissions
